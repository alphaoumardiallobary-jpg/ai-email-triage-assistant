# ai-email-triage-assistant
=======
# Asistente de IA para la clasificación de correos electrónicos Herramienta de IA que lee correos electrónicos y extrae información estructurada utilizando LangChain y OpenAI. 
## Funcionalidades - Clasifica la intención del correo electrónico - Detecta la prioridad - Extrae la fecha y la hora - Genera un resumen - Guarda los resultados en formato JSON 
## Ejemplo Correo electrónico de entrada: Hola, necesito cambiar mi cita del martes al jueves por la tarde. Salida: { "category": "meeting", "priority": "medium", "summary": "The sender wants to reprogramule a meeting.", "requested_action": "Reschedule appointment", "requested_date": "Thursday", "requested_time": "afternoon", "sentiment": "neutral" } 
## Tecnologías utilizadas Python LangChain OpenAI Pydantic
